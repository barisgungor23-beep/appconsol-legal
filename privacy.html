Privacy Policy

Last updated: March 28, 2026

Summary: AppConsol does not collect, store, or share any personal data. All your information stays on your device.

1. Overview

AppConsol ("the App") is a mobile application that connects to the Apple App Store Connect API to display your app sales and download data. This privacy policy explains how the App handles your information.

2. Data We Do NOT Collect

AppConsol does not collect, transmit, or store any personal data on external servers. Specifically:

  • No analytics or tracking SDKs are used
  • No user accounts are created
  • No personal information is collected
  • No data is shared with third parties for advertising or marketing
  • No cookies or tracking technologies are used

3. Data Stored on Your Device

The following data is stored locally on your device only:

  • API Credentials: Your App Store Connect API keys (Issuer ID, Key ID, Private Key, Vendor Number) are stored securely in the iOS Keychain. These credentials never leave your device.
  • User Preferences: Settings such as your preferred display currency and notification preferences are stored in local device storage (UserDefaults).
  • Cached Data: Sales report data fetched from Apple is cached temporarily in device memory to improve performance. This cache is cleared when the app is closed.

4. Third-Party Services

AppConsol connects to the following external services:

  • Apple App Store Connect API (developer.apple.com): Used to fetch your sales reports and app data. Communication is encrypted via HTTPS. Apple's privacy policy applies to their API service.
  • Exchange Rate API (open.er-api.com): Used to fetch current currency exchange rates for multi-currency display. No personal data is sent to this service — only a currency code is included in the request. No API key or user identifier is transmitted.

AppConsol does not use any analytics, crash reporting, or advertising services.

5. In-App Purchases

AppConsol offers premium features through in-app purchases processed by Apple. All payment processing is handled by Apple and is subject to Apple's Privacy Policy. AppConsol does not have access to your payment information.

6. Children's Privacy

AppConsol is a developer tool not intended for use by children under 13. We do not knowingly collect any information from children.

7. Data Security

Your API credentials are stored using the iOS Keychain, which provides hardware-level encryption. Credentials are marked as accessible only when the device is unlocked and are not included in device backups. All network communication uses HTTPS encryption.

8. Your Rights

Since AppConsol does not collect or store any personal data on external servers, there is no personal data to access, modify, or delete on our end. You can delete all locally stored data at any time by:

  • Deleting your API credentials from within the App settings
  • Uninstalling the App from your device

9. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated revision date. Continued use of the App after changes constitutes acceptance of the updated policy.
